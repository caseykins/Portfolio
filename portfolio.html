Here are a few of my projects:

Gotta Match 'Em All!

A fun take on the classic matching card game. This game was a solo project, built in 4 days, utilizing DOM manipulation in response to event listeners to flip cards upon being clicked, and judged against each other by Boolean values. If the player finds all the matches within the time allotted, they win!

JavaScript | HTML | CSS

Leafy Ledger

A web application designed to allow the user to find and research plants using the Trefle.io API. Once the user finds their desired plant, they can save the plant in their favorites and track each plant's watering schedule. For fun, we've also included the option to indicate if and when a plant has died, an unfortunate reality that all plant owners know all too well. Check out the plant cemetery page to see your late loved-ones and even have the ability to bring them back!

Express | EJS | Node.js | CSS | Bootstrap | HTML | Javascript | Sequelize | PostgreSQL | Passport.js | Axios

Brewser

A web application inspired by Untappd. Brewser allows users to search breweries based on location, using the OpenBreweryDB API and allows users to create beer reviews via check-ins.

Postgres | Express | React | Node.js | Axios | Sequelize | Passport.js | Sass

CardBoard

A web application based off of Trello, allowing users to create and customize digital to-do lists. Users can create any number of lists and list items, adding and deleting tasks as they go.

Express | EJS | Node.js | Sequelize | PostgreSQL | HTML | CSS | JavaScript
